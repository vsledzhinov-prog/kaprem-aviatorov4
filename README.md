# КапРем · Панель обследования — ул. Авиаторов, д. 4

Интерактивная веб-панель: шахматка дома, статистика по статусам и
прямой доступ к актам и презентациям по каждой квартире.

## Состав
```
index.html        — сама панель (открывается в браузере)
acts/kvNNN.pdf    — акты обследования (29 шт.)
reports/kvNNN.pdf — презентации-отчёты (17 шт.)
.nojekyll         — отключает обработку Jekyll на GitHub Pages
```

## Публикация на GitHub Pages

### Вариант A — отдельный репозиторий vsledzhinov-prog.github.io
1. Скопируйте всё содержимое этой папки в корень репозитория.
2. Сделайте commit и push.
3. В Settings → Pages выберите ветку (main) и папку / (root).
4. Через ~1 минуту панель будет по адресу:
   `https://vsledzhinov-prog.github.io/`

### Вариант B — в подпапку существующего сайта
1. Создайте в репозитории Pages папку, например `aviatorov4/`.
2. Скопируйте туда всё содержимое (index.html, acts/, reports/).
3. Ссылка: `https://vsledzhinov-prog.github.io/aviatorov4/`

## Как добавить квартиру
1. Положите PDF: акт → `acts/kvNNN.pdf`, презентацию → `reports/kvNNN.pdf`.
2. В `index.html` в начале `<script>` найдите объект `REPORTS` и добавьте:
   ```js
   "NNN": { "pres": "reports/kvNNN.pdf", "act": "acts/kvNNN.pdf" }
   ```
   Оставьте только те ключи (`pres` / `act`), для которых файл реально есть.

## Как это работает
Квартиры с документами помечены белой точкой и кликабельны.
Клик открывает меню: «Открыть презентацию» / «Открыть акт».
Если доступен только один документ — показывается только он.
